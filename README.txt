To compile the code simply type	"make" in the command line and then it will compile it all. To  run type "sint" into the command line and this will take you to the execution window. Type "./grayscott" to run the code. To change the input variables, head back to the login window using the command "exit" and use the command "emacs input.dat" and edit each line of this file accordingly. With the numbers representing: n, tau, T, F, k resp.
